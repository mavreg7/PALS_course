Bulk Enrollment — invite by CSV / Excel

Upload a CSV or Excel file of participants. Recognised columns: email (required), name (or first/last), role (student / instructor / lead_instructor — default student), optional cohort, optional phone (for WhatsApp / SMS invites). Not sure of the format? Click “Download template” for a ready-to-fill sheet. Each listed person can then self-register once using the link below. After uploading, use the “Email all invitees” button to send the invite — uploading alone does not email anyone.
